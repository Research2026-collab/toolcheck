# ToolCheck Inklusion – Anmeldeformular

Diese Version besteht aus einer einzigen `index.html` und kann direkt über GitHub Pages veröffentlicht werden.

## Wichtig

Diese Fassung nutzt `mailto:` statt E-Mail-Versand über das Mailprogramm. Beim Absenden öffnet sich das E-Mail-Programm der anmeldenden Person mit einer vorbereiteten Nachricht an:

`ev@keb-rheinland-pfalz.de`

Die Person muss die E-Mail im Mailprogramm noch absenden. Vorteil: Es wird kein externer Formularserver benötigt. Nachteil: Es ist nicht vollständig automatisch.

Für einen vollständig automatischen Versand braucht eine statische GitHub-Pages-Seite immer einen externen Dienst, z. B. Formspree, Web3Forms, Tally, Google Forms oder ein eigenes Backend.
