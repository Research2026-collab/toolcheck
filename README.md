# ToolCheck Inklusion – Anmeldeformular

Diese Version enthält die aktualisierte Veranstaltungsliste aus der zuletzt hochgeladenen CSV-Datei.

## Veröffentlichung über GitHub Pages

1. Die Datei `index.html` in das Repository `toolcheck` hochladen.
2. Eine bereits vorhandene `index.html` ersetzen.
3. Unter `Settings > Pages` sicherstellen, dass GitHub Pages für den Branch `main` und den Ordner `/root` aktiviert ist.
4. Die Seite ist danach voraussichtlich hier erreichbar:  
   https://research2026-collab.github.io/toolcheck/
